HSB 007 Observation Suite

Back to main lab page

This space is primarily used in conjunction with the facilities in the ChildLab. There are two fixed cameras, one in the main ChildLab, and one in the smaller testing room. Video and audio data can be recorded here.

Brief recording instructions

  1. Press the panic record button
  2. To stop the recording press it again
    3. To export the file Menu --> Archiving --> Log and Codec checked --> select the content to export --> Click every, then export Double Click --> tag Name to input a name
  3. Check Burn
